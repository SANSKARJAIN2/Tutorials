Read me file for firebase repository.
All the operations that are performed on the firebase in this repository is according to the official tutorial of firebase. 
Comments are according to me only and is based on my understanding of Firebase realtime database.
Link to the Tutorial :
https://firebase.google.com/docs/database/admin

All the dependencies are provided in the requirements.txt file in the repository.
You can install the dependencies by running the following command:

pip install -r requirements.txt

This created in a virtual environement and to create the virtual envirnment follow the steps ( it is assumed taht you have already installed python 3.6 > )

STEP1: open commond prompt

STEP2: navigate to the directory where the virtual environment has to be created.

STEP3: run the following commond in the commond prompt
	virtualenv --no-site-packages name of the environment

STEP4: To activate the environemt run the commond given below:
	name of the environemt/Scripts\activate

STEP5: To install the dependencies in virtual environment run
	pip install -r requirements.txt

Now you are all set to start working on firebase
